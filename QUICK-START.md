## Troubleshooting

**"Python is not installed" error:**
- Download Python from https://www.python.org/downloads/
- During installation, check "Add Python to PATH"
- Restart computer, then try again

**Install button doesn't appear:**
- Make sure you're using Chrome, Edge, or Safari
- Make sure you opened via `START.bat`/`START.sh` (not by double-clicking index.html)
- The URL should be `http://localhost:8000`, not `file://`

**Lost data:**
- Use Settings â†’ "ðŸ“‚ Import Backup" to restore from your JSON backup
- This is why regular backups are important!
